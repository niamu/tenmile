Playable Quotes for Game Boy

This project offers the ability to create and share playableplayable quotes** of [Game Boy](https://en.wikipedia.org/wiki/Game_Boy) games. A playable quote is a *durable*, *delimited* reference to a *specific* moment in a game along with a reference *recording* of how that moment can play out.
