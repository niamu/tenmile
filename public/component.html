<playable-quote> HTML Web Component

The following is a demonstration of the <playable-quote> HTML Web Component which wraps a valid <img> tag that has a src attribute of a playable PNG quote.

The brilliance of the playable quote encoded as a PNG is it provides an adequate fallback solution for when JavaScript fails to progressively enhance the image into an interactive piece of media. Additionally, the <img> remains accessible after the enhancement. So the quote PNG can be copied and shared while it is interactive.

Prerequisites

<playable-quote>

The <playable-quote> component has a JS module and related CSS file that should be placed in the <head> of your HTML document:


<script src="js/component.js" type="module" defer></script>
<link rel="stylesheet" type="text/css" href="css/component.css" />

Usage

The markup looks like this:


<playable-quote controls autoplay>
  <img alt="Playable Quote file for the Game Boy game called Tetris"
       src="images/80f5a65b-f7e3-4b40-b639-8e2c014de0ca-TETRIS-0d65-65740ab2.png">
</playable-quote>

  • controls

    The controls attribute allows for the user to take control of the playable quote using the "Take Control" button. If this attribute is not present, the button will not be available.

    Once a user clicks the "Take control" button their inputs are scoped to the <playable-quote> component and the border of image changes to green to indicate they are in control. When the component loses focus the pre-recorded gameplay begins again.

  • autoplay

    The autoplay attribute starts the pre-recorded playback immediately. If this attribute is not present a "Load" button is available to start the recorded playback.

Known issues/limitations

  • Unmuting a single component will unmute all loaded components on the page.
  • On-screen controls have not been added in yet.
  • Recording new quotes or playing ROMs is outside the scope for this component. The intention of this component is to provide an accessible and simple method for existing quotes to be shared.
