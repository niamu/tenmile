Playable Quotes for Game Boy

In the same way that you can quote a book, or make a clip of a video, this web page demonstrates a method to play, create, and share small moments from any Game Boy game.

As an example, try loading Tuff using the button.

Play Tuff for a bit, then try loading a quote of Tuff using the button.

The example quote demonstrates a very small part of Tuff. Initally you will see a replay of the inputs used to create the quote and if you press the Take control button, you'll be able to play that quote yourself.

However! If you try doing anything that wasn't part of that Playable Quote, the quote will reset.

This is a Playable Quote. Right now you are watching the movements that were made when the quote was created. However, if you press the Take control button, you can play the game too. Since this is only a quote, the game will reset if you try to do something that wasn't done when the quote was made.

Making your own Playable Quote: To make your own Playable Quote, use the button to upload any Game Boy game.

Play until you are ready to start you Playable Quote, then press the Record new quote button. Continue playing the game until you are ready to end your Playable Quote, then press the Stop recording button.

Once that is done, an image will appear below the emulator. This is a Playable Quote which you can download and share as you like.
